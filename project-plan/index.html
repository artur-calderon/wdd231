AprendaWeb – Website Plan

Site Name

AprendaWeb

The name was chosen because it clearly represents an educational platform focused on learning and web-based courses. It is simple, easy to remember, and suitable for an online learning guide.

Optional domain: aprendaweb.com

Site Purpose

The purpose of this website is to help users discover and explore online courses. The site will provide course listings with essential information such as category, level, duration, and ratings.

Scenarios

  • Which beginner-friendly online courses are available?
  • How can I view details about a specific course?

Color Scheme

  • Primary Color (#1e88e5): Used for headings, header background, and accents.
  • Secondary Color (#f5f7fa): Used for page background and spacing areas.

Typography

  • Montserrat: Used for headings and titles.
  • Roboto: Used for body text.
